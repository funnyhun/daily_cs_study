# 선택 정렬(Select Sort)

선택 정렬은 정렬되지 않은 정수 배열을 입력으로 받았을 때, 배열을 순회하면서 가장 작은 값을 찾은 뒤 탐색 범위의 시작점으로 이동시키는 알고리즘이다. <br>
이때, 정렬된 요소의 수만큼 탐색 범위의 시작점을 우측으로 이동시켜 오름차순으로 정렬되게 된다. <br>
해당 알고리즘의 시간 복잡도는 O(N^2)로 이차 시간 알고리즘에 해당된다.

구체적으로 선택 정렬은 비교와 스왑 연산으로 이루어 진다. <br>
여기서 같은 카테고리인 버블 정렬과의 차이점이 생겨나는데 버블 정렬은 같은 수의 비교와 연산이 일어나는 반면 선택 정렬은 각 정렬 루프(비교는 동일)에서 1번의 스왑이 일어난다. <br>
물론, 위의 예시는 최악의 시나리오를 기반으로 한다.

- 1회차 : N - 1번
- 2회차 : N - 2번
- ...
- N회차 : 1번

즉, 비교 연산 횟수는 (N - 1) + (N - 2) + ... + 1 = N(N - 1) / 2 이다. <br>
이때, 선택 정렬의 스왑은 각 루프에서 1회이므로 동작 단계는 N(N - 1) / 2 + (N - 1)이다. <br>
이를 전개하면, (N^2 + N - 2) / 2이다. <br>
결과적으로, 선택 정렬의 시간 복잡도는 O(N^2)이다.
